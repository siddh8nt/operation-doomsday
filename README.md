# OPERATION DOOMSDAY
**Strategic Nuclear Escalation Simulator**

A high-stakes, browser-based interactive simulation that places you in the Situation Room during a 15-minute nuclear crisis. As the Prime Minister, you must navigate a complex decision tree against an adversarial intelligence.

## Overview
Early warning systems indicate a ballistic missile launch. You have exactly 15 minutes to impact. Connected to the "Adversarial Projection Network" (APN), every decision you make is evaluated by a Minimax game-theory simulation predicting the enemy's worst-case response.

Maximize the survival of your nation while minimizing the enemy's strategic advantage. Follow the AI's mathematically optimal path, or override it based on your human instinct.

## Features
*   **Real-Time & Turn-Based Pressure:** A global 15-minute timer ticks down in real-time, but massive chunks are deducted based on the time cost of your strategic choices.
*   **Advanced AI Engine:** A functional Minimax algorithm with Alpha-Beta pruning calculates the optimal path, showing its "thought process" and pruned branches.
*   **Interactive Tree Visualizer:** Watch the APN traverse the decision tree step-by-step using smooth Framer Motion animations.
*   **Hyper-Realistic Scenarios:** Deeply researched, high-stakes military and geopolitical trade-offs. 
*   **Military HUD Aesthetic:** Cold, intense, terminal-style UI built with Tailwind CSS.

## Tech Stack
*   **Frontend:** React / Next.js
*   **Styling:** Tailwind CSS
*   **Animations:** Framer Motion
*   **Language:** TypeScript

## Getting Started
1. Clone the repository:
   `git clone https://github.com/siddh8nt/operation-doomsday.git`
2. Install dependencies:
   `npm install`
3. Start the development server:
   `npm run dev`

This is a work of interactive fiction and simulation. The scenarios, institutions, and outcomes are simulated game mechanics and do not reflect real-world operational protocols or classified military doctrines.
